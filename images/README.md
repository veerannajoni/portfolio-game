# Images folder

Add these files for favicons and social sharing:

| File | Purpose |
|------|--------|
| `favicon.ico` | Browser tab icon (e.g. 32×32) |
| `favicon-192.png` | PWA / Android icon (192×192) |
| `apple-touch-icon.png` | iOS home screen icon (180×180) |
| `og-image.png` | Social share image for LinkedIn, Twitter, etc. (1200×630 recommended) |

You can generate favicons from a single image using [realfavicongenerator.net](https://realfavicongenerator.net/) or similar. For `og-image.png`, use any image editor or Canva to create a 1200×630 image with your name and “Quest for the Code”.

Until these files exist, the browser may show 404s for the icon links; the site will still work.
